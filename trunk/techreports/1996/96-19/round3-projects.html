This appears to require only two projects:

1. TRLtoHTML
The TRLtoHTML class takes a TRLLog object and a file name as input and writes out the TRLLog entries in HTML format. It includes a "summary" section at the bottom with various statistics about the time log entries.

2. TRLLogGUI
Provides the GUI "manager" that displays the contents of a TRL Log and allows editing to be performed on each entry.
