Access Control

AEN's access control mechanisms implements three forms of access to individual nodes: read access, write access, and annotation access. Users can set each form of access for individual users or for all users. When a user has read access for a specific node, they can retrieve and view it. Similarly, when a user has annotation access they can comment on the node and make links from the node, but not edit its contents. Finally, when a user has write access to a node they can change the text in the node. Each of these access mechanisms are independent. When the owner wants another user to read the node they can set the read access so that the other user can read the node. The owner may decide that another user should have write access but not annotation access.
