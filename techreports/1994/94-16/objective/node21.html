Related Work

  AEN's set of features were designed to explore the issues presented in the introduction. AEN provides a typed hypertext document, dynamic user defined table of contents, real-time communications with other users of the system, real-time knowledge of what the other users of the system are doing and access control for each node. Other hypertext or collaborative editors provide different combinations of these features. SEPIA[4] is a collaborative hypermedia authoring environment. It does not have a dynamic table of contents. Note: I have not looked into the following systems HyperCard does not support collaboration. WWW [1] does not provide the user with knowledge of other users. ShrEdit [7]? GROVE [2]? Quilt [3]?
