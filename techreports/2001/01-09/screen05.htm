ICS CourseApp Request Page

This page currently displays only those courses for which you are qualified to request an approval code at this time.  To display all ICS courses along with explanations why you are not qualified for certain courses, click here. 

Use the pull-down list in the last row to specify up to four ICS courses as your choices. Use "1" to indicate your first choice, "2" to indicate your second choice, and so forth.  When you are done specifying your choices, you can submit them all for processing using the "submit" button at the bottom of the page. 

Note: If you request two courses with the same choice value, your request will be returned for correction.  If you have a prior request at a given choice value you must delete it by going to your Student Status page before submitting a new request with that same choice value.

For courses with multiple sections, use the "exclude" checkbox in the last column if there is a section you absolutely cannot attend. This will ensure you are not given an approval code for that section.
