This directory contains subdirectories with LaTeX styles used for CSDL technical reports. 

IEEE_CS_Latex:
  The style file for IEEE conference publications such as ICSE. 
  TEXINPUTS: csdl-techreports/styles/IEEE_CS_Latex 
  BSTINPUTS: csdl-techreports/styles/IEEE_CS_Latex 

svjour3:
  The style for Springer-Verlag submissions (Journal of Automated Software Engineering)
  TEXINPUTS: csdl-techreports/styles/svjour3/global/
  BSTINPUTS: csdl-techreports/styles/svjour3/bibtex/

natbib:
  The style file for the 'natbib' bibliography style used by ASE.
  Included with the TeX Live distribution. 
  No need to put on your environment variables, included for documentation purposes.
